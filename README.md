# Max/MSP Project Templates Course Repository

A comprehensive collection of Max/MSP project templates and examples designed for digital arts students learning interactive audiovisual programming with Max 8+.

## Course Description

This repository provides base structures and templates for creating interactive compositions, audiovisual processing systems, and multimedia prototypes using Max/MSP. The course focuses on:

- **Project reuse and modular organization**: Learn how to structure Max projects for maintainability and collaboration
- **Documentation best practices**: Develop clear, well-documented patches that others can understand
- **Interactive audiovisual programming**: Build real-time systems for sound synthesis, audio processing, and visual generation
- **Digital arts workflow**: Establish professional habits for creative coding projects

Perfect for students and digital artists exploring creative coding with Max/MSP.

## Requirements

- Max 8 or higher
- Basic understanding of digital signal processing concepts
- Audio interface (recommended for audio projects)

## Quick Start

1. **Clone this repository**:
   ```bash
   git clone https://github.com/Rennovatio04/Max.git
   cd Max
   ```

2. **Open a template**:
   - Navigate to the `templates/` folder
   - Double-click any `.maxpat` file to open it in Max
   - Save your work with a new name to preserve the original template

3. **Explore examples**:
   - Check the `examples/` folder for complete project demonstrations
   - Study the patches to understand different techniques and approaches

4. **Add your audio files**:
   - Place your audio samples in the `audio/` folder
   - Reference them in your patches using relative paths

## Working with Templates

Templates in the `templates/` folder provide starting points for common Max/MSP tasks. To use a template:

1. Copy the template file
2. Rename it for your project
3. Customize the patch for your specific needs
4. Document your modifications with comments
